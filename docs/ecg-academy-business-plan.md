# 「心电学堂」投资制作计划书

> 基于 EP Mentor 现有资产（AI 病例生成管道 + 苏格拉底教学引擎 + 84 个已修复 UI 组件）转型为一款面向医生群体的心电图判读 AI 教学产品。

---

## 一、产品架构

### 1.1 三层内容金字塔

```
                    ┌─────────────────────────┐
                    │  EP 进阶（品牌制高点）     │   ~120 病例（已有）
                    │  心内电生理 + 复杂消融     │   面向 EP 专科医生
                    │  免费，用于吸引权威用户     │
                    ├─────────────────────────┤
                    │  心律失常（核心收费层）     │   新增 80 个病例
    💰 付费          │  AF / AFL / SVT / VT      │   面向心内科 + 急诊科医生
                    │  AV 阻滞 / 束支阻滞         │   主力变现
                    ├─────────────────────────┤
                    │  基础心电图（引流层）        │   新增 120 个判读案例
    🆓 免费          │  P-QRS-T 分析 / 电轴 /     │   面向所有医生 + 医学生
                    │  心梗定位 / 电解质异常       │   最大用户池
                    └─────────────────────────┘
```

## 五、获客策略

### 5.1 零成本获客渠道

| 渠道 | 方法 | 预期效果 |
|------|------|------|
| **微信群** | 心内科/急诊科/规培群分享免费心电图挑战 | 种子用户 200+ |

| **科室推荐** | 给老师/科主任试用，让他们推荐给进修生 | 机构订单入口 |
| **医学会议** | 相关会议展台/演讲提一下工具 | 精准获客 |

### 5.2 核心增长引擎：每日 ECG 挑战

```
用户收到推送 → 看图选答案 → 查看 AI 解析 → 分享给同事讨论
     ↑                                              ↓
  明天再来 ← 看到正确率排名 ← 同事也点进来试
```

这个循环是**免费的增长飞轮**，不需要广告费。

## 六、风险与应对

| 风险 | 概率 | 影响 | 应对 |
|------|:---:|:---:|------|
| AI 回答不准确，医疗风险 | 中 | 🔴 高 | 每页标注「仅供教学，不构成临床决策」；内容人工审核 |
| DeepSeek API 涨价 | 低 | 🟡 中 | 锁定 `deepseek-chat` 价格；可选备用模型 |
| 微信审核不通过（医疗类目） | 中 | 🔴 高 | 提前准备《增值电信业务许可证》或走「教育-职业教育」类目 |
| 竞品出现 | 中 | 🟡 中 | 先发优势 + AI 教学体验壁垒；竞品多是题库，少有 AI 对话 |
| 版权风险（现有病例） | 低 | 🟡 中 | 基础心电图内容自建；现有病例标注出处并注明「仅供学习参考」 |
| 用户增长缓慢 | 高 | 🟡 中 | 内容驱动增长（SEO + 公众号）需要耐心，6 个月不算慢 |
